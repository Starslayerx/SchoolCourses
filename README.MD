# 使用

1. 把代码中**你的账号**、**你的密码**换成自己的账号密码
2. 根据自己的浏览器版本下载相应驱动(文件中驱动为chromedriver81版本)
3. 运行代码，如果报错请自行调试
4. 目前存在bug，如果页面暂停请重新运行
5. 运行足够时间后记得**手动退出**学习页面，否则系统不会计时
